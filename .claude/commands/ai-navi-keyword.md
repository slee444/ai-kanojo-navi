# AI彼女ナビ SEOキーワード選定

AI彼女ナビのSEO戦略に基づき、次に書くべき記事のキーワードを選定してください。

## 手順

1. `PROJECT.md` を読んでサイトの戦略・ターゲット・方針を確認する
2. `content/` 以下の既存記事をすべて確認し、既にカバー済みのトピックを把握する
3. 以下の観点でキーワード候補を10個挙げる：
   - 検索ボリュームが多そうなもの優先（月間1,000〜10,000程度）
   - ターゲット（20〜30代男性、AI彼女を探している初心者〜比較検討層）に刺さるもの
   - 既存記事と重複しないもの
   - SEO戦略（主要KW→比較系→ロングテール）の順序に沿ったもの
   - 記事タイプ（articles/reviews/compare）も明示する

4. 候補10個を表形式で提示（キーワード・記事タイプ・想定検索ボリューム・理由）
5. その中から最も優先度が高い**1つを推薦**し、以下を提示する：
   - 推薦キーワード
   - 記事タイトル案（3案）
   - 記事の構成案（h2見出し5〜7本）
   - 狙うべき検索意図

記事を書く準備ができたら `/ai-navi-write` を使ってください。
